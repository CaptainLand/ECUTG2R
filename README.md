## 关于 G2R

G2R 是东华理工大学的音游同好社群。我们因为同一拍节奏相遇，在街机前交流手法、研究谱面，也通过跨游戏、跨年级的联合手元记录共同的游玩经历。

社群覆盖的音游包括 **maimai DX、Phigros、Arcaea、Lanota、DEEMO、Dynamix、Malody、osu!、CHUNITHM、SOUND VOLTEX** 等。无论设备、水平或主要游玩平台，只要喜欢节奏游戏，都欢迎加入。

## 网站介绍

这是 G2R 的轻量级静态介绍网站，采用明亮的棱镜色彩和街机音游选曲界面设计。

- 响应式首页，兼容桌面端和移动端
- 无缝滚动的音游标题条幅
- 社群联合手元选曲列表
- Bilibili 作品直达链接
- QQ 群号一键复制
- 支持系统“减少动态效果”偏好
- 不依赖框架、构建工具或外部图片资源

## 本地预览

项目是纯静态页面，不需要安装依赖。克隆仓库后直接打开 `index.html`，或在项目目录启动任意静态文件服务器：

```powershell
python -m http.server 8000
```

随后访问 `http://localhost:8000/`。
